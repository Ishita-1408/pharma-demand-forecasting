## 📌 Overview

A time-series machine learning system that forecasts monthly drug demand for PIL's pharmaceutical product portfolio. The system helps optimize **inventory planning**, reduce **stockouts**, and improve **production scheduling** across 5 regions of India.

**Key Results:**
- Forecasts demand for 8 pharmaceutical products across North, South, East, West, and Central India
- Average MAPE < 10% across all products using Facebook Prophet
- Identifies seasonal demand peaks (winter for antidepressants, Q4 hospital restocking)
- Interactive Streamlit dashboard with downloadable forecast CSVs

## 📊 Dataset

The dataset simulates PIL's internal sales records (anonymized for confidentiality):

| Column | Description |
|--------|-------------|
| `product_name` | 8 PIL-style pharmaceutical products |
| `category` | Psychotropic / Antidepressant / Anxiolytic / Antipsychotic |
| `region` | North / South / East / West / Central |
| `units_sold` | Monthly units sold (target variable) |
| `revenue_inr` | Monthly revenue in INR |
| `stockout_flag` | 1 if regional stock fell below threshold |
| `lag_1_units`, `lag_3_units` | Lag features for ML |
| `rolling_avg_3` | 3-month rolling average |

## 🔍 Key Findings

1. **Antidepressants (Escitab-10, Fluox-20)** show strong winter seasonality — demand peaks ~18% above average in Nov–Feb
2. **October Diwali dip** affects all products (~8% drop) due to logistics slowdown
3. **North region** accounts for ~30% of total demand — priority region for inventory
4. **Q4 hospital restocking** creates a predictable demand spike in Nov–Dec
5. Prophet outperforms SARIMAX on products with strong seasonality; SARIMAX is competitive on stable psychotropics
